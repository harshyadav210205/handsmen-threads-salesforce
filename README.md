# HandsMen Threads 👔  
### Elevating the Art of Sophistication in Men's Fashion  

## 📌 Overview  
HandsMen Threads is a Salesforce-based project designed to improve customer experience and streamline business operations in the fashion industry.

The system focuses on:
- Strong data modeling  
- Data integrity & validation  
- Business process automation  

---

## 🚀 Features  

### ✅ Automated Order Confirmations  
Customers receive email notifications after placing orders.

### 🎯 Dynamic Loyalty Program  
Customer loyalty status updates automatically based on purchase history.

### 📦 Proactive Stock Alerts  
Warehouse team is notified when stock drops below 5 units.

### ⏰ Scheduled Bulk Order Processing  
Midnight batch job updates inventory and financial records.

---

## 🛠 Tech Stack  
- Salesforce Apex  
- Lightning App Builder  
- Flow Builder  
- Apex Triggers  
- Asynchronous Apex

## 🧠 Concepts Used
